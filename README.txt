راهنمای تبدیل به اپ اندروید (ساده‌ترین راه)
=============================================

مرحله ۱ — گرفتن کلید API
--------------------------
1. برو به console.anthropic.com و ثبت‌نام کن (اگه نداری)
2. یه API key بساز (بخش API Keys)
3. یادت باشه: این کلید هزینه‌ی مصرفت رو از حساب خودت کم می‌کنه. فقط برای استفاده‌ی شخصی خودت نگهش دار، جایی پابلیکش نکن.

مرحله ۲ — آپلود روی GitHub Pages (رایگان)
--------------------------
1. یه حساب github.com بساز (اگه نداری)
2. یه ریپازیتوری جدید بساز (مثلا اسمش: my-ai-app) و public بذارش
3. سه فایل داخل این پوشه (index.html, manifest.json, icon-192.png, icon-512.png) رو آپلود کن
   (از دکمه "Add file" > "Upload files" توی گیت‌هاب)
4. برو به Settings > Pages، و از قسمت Source شاخه‌ی main رو فعال کن
5. بعد چند دقیقه، سایتت روی یه آدرس مثل این بالا میاد:
   https://USERNAME.github.io/my-ai-app/

مرحله ۳ — ساخت APK با PWABuilder
--------------------------
1. برو به pwabuilder.com
2. آدرس سایتت از مرحله‌ی قبل رو بده (https://USERNAME.github.io/my-ai-app/)
3. بزن Start، صبر کن تا اسکن کنه
4. برو روی تب Android و Generate Package بزن
5. فایل APK یا AAB دانلود میشه

مرحله ۴ — نصب روی گوشی
--------------------------
- فایل APK رو به گوشیت منتقل کن (تلگرام به خودت، ایمیل، یا کابل)
- روی فایل بزن، اگه گفت "منابع ناشناس" رو فعال کن، بزن نصب

نکته مهم
--------------------------
اولین بار که اپ رو باز کردی، از دکمه‌ی چرخ‌دنده بالا کلید API‌ت رو وارد و ذخیره کن.
حافظه و تاریخچه‌ی گفتگو فقط روی همین گوشی ذخیره می‌شه.
